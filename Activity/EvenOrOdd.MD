## Duration
30 minutes

## Challenge
Write a method that prints "Even" if the number passed is even and "Odd" if the number is odd.
  
## Examples
```java
//Sample Input
evenOrOdd(1156); 

//Expected Output
Even
```

```java
//Sample Input
evenOrOdd(2433);

//Expected Output
Odd
```

**HINT:** look into the % operator
